# 一个全套公司企业官网 前端项目 （springboot +vue ）

[后端项目地址](https://github.com/liyinchigithub/company_official_website_server)

## 技术栈

- nodejs
- vue
- elementUI


## 安装
```
yarn
```

### 启动
```
yarn dev
```

### 打包
```
yarn build
```
